# 词表生成任务

## 角色定位

请作为政务服务领域专家，为每个意图提取特征词并打分。

## 提取规范

根据「意图名称」和示例问题，按以下六步提取特征词：

1. **智能分词**（segments）：对意图名称分词，保持专业术语完整性
2. **动宾识别**（object/action）：从分词集合中识别动词和宾语
3. **核心词**（core）：意图名称中最关键的词，区分度最高
4. **重要词**（important）：业务细节词，与核心词不重复
5. **相关词**（related）：用户咨询时可能使用的场景描述词
6. **同义词**（synonyms）：核心词和重要词的近义词、简称、口语表达

## 打分规则

| 权重范围 | 判断标准 | 示例 |
|----------|----------|------|
| 0.95-1.0 | 专业术语，无歧义 | 失业保险金、稳岗返还 |
| 0.85-0.95 | 常用表达，歧义很小 | 失业金、领失业金 |
| 0.75-0.85 | 通用动作词，有一定歧义 | 申请、领取、办理 |
| 0.60-0.75 | 宽泛表达，歧义较大 | 怎么办、想要 |

## 输出格式

请严格按以下JSON格式输出：

```json
{
  "意图映射表": {
    "意图名称1": {
      "segments": ["失业保险金", "申领"],
      "object": "失业保险金",
      "action": "申领",
      "keywords": {
        "core": ["失业保险金", "申领"],
        "important": ["失业人员", "申请", "领取"],
        "related": ["失业证明", "就业登记"]
      },
      "synonyms": {
        "失业保险金": ["失业金", "失业补助金"],
        "申领": ["申请", "办理", "领取"]
      }
    }
  },
  "词权重表": {
    "失业保险金": {"权重": 0.98, "理由": "专业术语，无歧义"},
    "失业金": {"权重": 0.92, "理由": "常用简称"}
  }
}
```

## 注意事项

1. **同一个词在不同意图下使用相同权重**
2. 参考示例问题理解用户表达习惯
3. 提取口语化表达作为相关词
4. 避免提取过于宽泛、歧义大的词
