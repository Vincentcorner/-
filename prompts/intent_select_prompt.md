# 意图筛选任务

请从候选意图中选择最匹配用户诉求的意图。

## 选择规则

1. **语义最匹配**：选择与用户表达语义最接近的意图
2. **得分参考**：得分高的意图优先考虑，但不是唯一标准
3. **上下文理解**：考虑用户可能的真实需求
4. **单一选择**：只选择一个最匹配的意图

## 判断依据

- **命中词的相关性**：命中的词是否与用户诉求语义强相关
- **意图的完整性**：选择的意图是否能完整回应用户需求
- **常见场景**：用户通常在什么情况下会这样表达

## 输出格式

请按以下JSON格式输出：

```json
{
  "选中意图": "意图名称",
  "置信度": 0.95,
  "理由": "简要说明选择理由"
}
```

## 置信度说明

| 置信度 | 含义 |
|--------|------|
| 0.9-1.0 | 非常确定，语义完全匹配 |
| 0.7-0.9 | 较为确定，语义基本匹配 |
| 0.5-0.7 | 一般确定，可能存在歧义 |
| <0.5 | 不确定，建议人工复核 |

## 示例

**用户诉求**：我想领失业金

**候选意图**：

1. 失业保险金申领（得分: 0.95）
2. 失业保险金标准查询（得分: 0.70）
3. 失业登记（得分: 0.45）

**输出**：

```json
{
  "选中意图": "失业保险金申领",
  "置信度": 0.98,
  "理由": "用户明确表达"领"的意图，与"申领"语义完全匹配"
}
```
