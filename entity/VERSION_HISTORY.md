# 版本历史 (Version History)

## v2.1.0 (2026-01-19)
- **Feature**: 引入 Agent 驱动的实体关系智能总结，取代基于规则的简单复述。
- **Fix**: 修复 Excel 报告中"具体条款内容"列被截断的问题。
- **Opt**: 优化输出目录结构，统一使用 `result/`。

## [2.1.0] - 2026-01-19

### 变更
- **目录结构重组**
  - `data/input/` → `originalfile/`（原始文件目录）
  - `data/output/` → `result/`（结果输出目录）
  - `CHANGELOG.md` → `VERSION_HISTORY.md`（版本历史文件）
- 更新所有脚本和文档中的路径引用

### 新目录结构
```
entity/
├── .agent/           # Agent 配置
├── changelog/        # 变更日志
├── originalfile/     # 原始文件
├── result/           # 分析结果
├── scripts/          # 脚本
├── VERSION_HISTORY.md
└── README.md
```

## [2.0.0] - 2026-01-19

### 新增
- 智能关系总结生成策略，取代模板化输出

### 优化
- `generate_summaries.py` 重构：
  - 三层回退机制：实体角色分析 → 关键词提取 → 条款核心描述
  - 支持灵活就业人员、失业人员、用人单位、社保机构、税务机关、平台等实体
  - 自动从条款内容提取核心句子
- 工作流文档更新，增加智能策略说明

### 示例对比
| 优化前 | 优化后 |
|--------|--------|
| 灵活就业人员与参加在该条款中存在业务关联 | 灵活就业人员参加失业保险遵循自愿原则 |
| 灵活就业人员与失业保险待遇在该条款中存在业务关联 | 符合条件的灵活就业人员可享受失业保险待遇 |
