# 主题分析提示词 v2.0

## 任务说明

读取 `analysis_data.json`，对每个条款进行智能主题分析，生成结构化的关系总结。

---

## 分析要点

对每个条款分析以下内容：

### 1. 条款主题
这条规定的**核心内容**是什么？用简短的名词短语概括（如"基金支出范围"、"领取条件"）。

### 2. 核心实体识别
哪些实体与主题**直接相关**？

### 3. 实体角色判断

| 角色 | 说明 | 示例 |
|------|------|------|
| 主体 | 条款规范的核心对象 | 失业保险基金用于支付... |
| 客体 | 规范行为的对象 | ...支付失业保险金 |
| 义务主体 | 承担义务的实体 | 用人单位应当缴纳... |
| 权利主体 | 享有权利的实体 | 失业人员可以领取... |
| 受益对象 | 从规定中获益的实体 | ...保障失业人员权益 |
| 执行主体 | 负责执行的实体 | 社会保险经办机构负责... |
| 监管主体 | 负责监管的实体 | 社会保险行政部门依法监督... |
| 定语修饰 | 仅作为定语出现 | 职工基本医疗保险费（边缘） |

### 4. 关联程度判断

| 程度 | 说明 | 处理方式 |
|------|------|----------|
| 核心 | 条款主要针对该实体 | 必须生成总结 |
| 次要 | 条款涉及但非主要 | 生成简要总结 |
| 边缘 | 仅作为修饰语出现 | 标注但可跳过 |

### 5. 关系总结生成

> **重要**：每条 summary **必须包含该实体名称和相关实体**，使关系一目了然。

✅ 正确：`"灵活就业人员应依法缴纳失业保险费"`  
❌ 错误：`"应依法缴费"`

保存到 `relation_summaries.json`：

```json
[
  {
    "article": "第十二条",
    "topic": "失业保险基金支出范围",
    "entities": [
      {
        "name": "失业保险基金",
        "role": "主体",
        "relevance": "核心",
        "summary": "失业保险基金用于支付失业保险金等六类待遇"
      },
      {
        "name": "失业人员",
        "role": "受益对象",
        "relevance": "次要",
        "summary": "失业人员可从失业保险基金中获得保险金、医疗补助等待遇"
      },
      {
        "name": "职工",
        "role": "定语修饰",
        "relevance": "边缘",
        "summary": "职工基本医疗保险费由失业保险基金支付（职工仅作定语）"
      }
    ]
  }
]
```
