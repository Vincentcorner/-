# 意图筛选任务

请从候选意图中选择最匹配用户诉求的意图。

## 选择规则

1. **语义最匹配**：选择与用户表达语义最接近的意图
2. **上下文理解**：考虑用户可能的真实需求

## 判断依据

- **意图的完整性**：选择的意图是否能完整回应用户需求
- **常见场景**：用户通常在什么情况下会这样表达

## 输出格式

请按以下JSON格式输出：

```json
{
  "选中意图": "意图名称",
  "理由": "简要说明选择理由"
}
```

## 示例

**用户诉求**：我想领失业金

**候选意图**：

1. 失业保险金申领
2. 失业保险金标准查询
3. 失业登记

**输出**：

```json
{
  "选中意图": "失业保险金申领",
  "理由": "用户明确表达"领"的意图，与"申领"语义完全匹配"
}
```
