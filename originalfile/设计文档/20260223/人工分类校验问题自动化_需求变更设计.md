## 1. 变更概述

在意图匹配 Web UI 的最终结果展示区域新增三项功能：

1. **标题行**：结果表格上方添加"🔍 人工分类校验问题自动化"标题

3. **详情按钮 + 弹窗**：每行数据可点击查看详细诊断分析和匹配明细

## 2. 自动化校验流程

### 2.1 流程示意图

```
结果详情信息
    ↓
┌─────────────────────┐
│ 意图得分清单中       │──── 是 ──→ 大模型筛选问题
│ 是否有标杆意图？     │
└─────────────────────┘
    │ 否
    ↓
┌─────────────────────┐
│ 权重表中是否有       │──── 否 ──→ 权重分表意图不全
│ 相关标杆意图？       │
└─────────────────────┘
    │ 是
    ↓
  计算标杆意图权重分
    ↓
┌─────────────────────┐
│ 是否权重分低？       │──── 否 ──→ topK意图取得不全
└─────────────────────┘
    │ 是
    ↓
  特征词匹配不足
  （转写未覆盖特征词或权重分过低）
```

### 2.2 诊断结果类型

| 类型 | 含义 | 建议措施 |
|------|------|----------|
| ✓ 一致 | 算法意图与标杆意图匹配 | 无需处理 |
| 大模型筛选问题 | 标杆意图在候选列表中但AI未选中 | 优化意图筛选提示词 |
| 权重分表意图不全 | 标杆意图不在权重词表中 | 补充权重词表中的意图映射 |
| topK意图取得不全 | 标杆意图有得分但未进入候选列表 | 调大 `top_k` 参数 |
| 特征词匹配不足 | 标杆意图在词表中但未匹配到特征词 | 补充转写策略或调整特征词 |

## 3. 详情弹窗内容

点击表格中"📋 详情"按钮，弹窗展示以下分区信息：

### 3.1 诊断分析区

- 以醒目样式展示诊断结果（一致=绿色框，不一致=橙色警告框）

### 3.2 基本信息区

| 字段 | 说明 |
|------|------|
| 原始问题 | 完整的用户诉求原文（不截断） |
| 转写结果 | 三维度转写全文 |
| 算法意图 / 标杆意图 | 对比展示 |
| 置信分 / 转写来源 / 结果 | 辅助信息 |

### 3.3 意图得分明细表

展示所有候选意图的得分、命中词数和命中特征词，便于对比分析。

### 3.4 补充信息

- 意图特征词（格式：`意图名（词1、词2）`）
- 全量意图列表
