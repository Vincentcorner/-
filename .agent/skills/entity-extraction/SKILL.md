# 法律法规主体实体提取技能 v2.0

## 快速命令

| 命令 | 说明 | 本地/模型 |
|------|------|-----------|
| `/extract <文件名>` | 完整流程：解析 → 识别 → 总结 → 报告 | 混合 |
| `/analyze <文件名>` | 仅解析+识别（不消耗 token） | 本地 |
| `/summarize` | 生成关系总结（需模型参与） | 模型 |
| `/intent <法规名>` | 生成实体意图清单（需模型参与） | 模型 |
| `/report` | 生成 Excel 报告 | 本地 |

## 工作流程

```
┌─────────────────────────────────────────────────────────────────┐
│ /analyze（本地）                                                  │
│  文档解析 → 条款拆分 → 实体识别 → 词频统计                          │
└───────────────────────────┬─────────────────────────────────────┘
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│ /summarize（模型）                                                │
│  读取实体关系 → Agent 生成一句话业务总结                            │
└───────────────────────────┬─────────────────────────────────────┘
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│ /report（本地）                                                   │
│  合并数据 → 生成 Excel 报告                                        │
└─────────────────────────────────────────────────────────────────┘
```

## 能力描述

本技能用于从法律法规文件（Word/PDF）中提取所有主体类实体，包括：
- 自然人（公民、当事人、法定代表人等）
- 企业法人（XX公司、XX集团等）
- 政府机关（国务院、XX部、XX局等）
- 司法机关（人民法院、人民检察院等）
- 社会组织（XX协会、XX基金会等）
- 事业单位（XX学校、XX医院等）

## 主体实体排序原则

> **核心原则**：按照法律法规的立法目的，对主体实体进行分层排序

| 优先级 | 主体类型 | 说明 | 示例 |
|--------|----------|------|------|
| **1** | 核心保护群体 | 法规核心保护的权利主体 | 失业人员、消费者、患者、劳动者 |
| **2** | 义务主体 | 承担主要义务的主体 | 用人单位、企业、经营者、医疗机构 |
| **3** | 服务执行主体 | 执行法规的服务机构 | 经办机构、服务中心、鉴定机构 |
| **4** | 监管主体 | 监督管理的政府机关 | 行政部门、审计机关 |

## 本地 vs 模型 职责划分

### 本地脚本（不消耗 token）
- 文档解析 (`parse_document.py`)
- 条款拆分 (`article_parser.py`)
- 实体识别 (`entity_analyzer.py`)
- 词频统计与条款定位
- Excel 报告生成 (`generate_excel.py`)

### 模型职责
- 关系总结生成（基于条款内容生成业务关系描述）
- 复杂实体识别（规则无法覆盖的新实体）
- 需求理解与分析
